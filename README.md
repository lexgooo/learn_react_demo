## 问题收集
- 在 `click` 事件中使用 `this.setState({value: 'X'})` 来设置 `value` 值，为什么不能直接赋值，形如 `this.state.value = 'X'`

## 最佳实践
- 通常会将代表事件的监听 `prop` 命名为 `on[Event]`，将处理事件的监听方法命名为 `handle[Event]` 这样的格式。
